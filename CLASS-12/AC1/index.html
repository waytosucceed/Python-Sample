A dictionary is a collection of key and value pairs and is often referred to as a dict in Python.

A key:value pair is referred to as an item in a dictionary.

Creating a Dictionary

To create an empty dictionary, you can simply write the name you would like to give to your dictionary followed by ‚Äò=‚Äô and a pair of curly brackets {}.


Now let‚Äôs take the example of two students: Rita and Reena. Rita scored 50 and Reena scored 20 in a test. If you wish to store their names along with the marks they‚Äôve scored, you can do it like so:


my_data = {'Rita': 50, 'Reena': 20}
In this example: - my_data is a dictionary, - Rita and Reena are keys, and - 50 and 20 are values.

There are certain rules for naming the keys in a dictionary:

üí° Rules for keys in the dictionary:

üîë You can‚Äôt have the same keys for multiple values. If you use the same keys for multiple values, the subsequent entries will overwrite the previous ones.

For example:

You can see that when we try to add a second entry with the key 'Rita': my_data = {'Rita': 50, 'Reena': 20, 'Rita': 10} When we print it, the second occurrence of 'Rita': 10 overwrites the first occurrence of 'Rita': 50, so the dictionary will contain:

{'Rita': 10, 'Reena': 20}

üîë Only strings, numbers, and tuples can be used as keys. If you use a list as a key, you will get a TypeError.

Examples of valid keys:

1. Number as keys: my_data = {1: 'one', 2: 'two'}
2. String as keys: my_data = {'Rita': 50, 'Reena': 20}
3. Tuple as keys: my_data = {(1, 2): 'point1', (3, 4): 'point2'}

üìù Note: You can use any data type for values.

We have already used strings and integer numbers as values in our previous examples. You can even have floats, lists, and tuples as values.

Examples of valid values:

- List as values: my_data = {'Rita': [50, 60], 'Reena': [20, 30]}

- Tuples as values: my_data = {'Rita': (50, 60), 'Reena': (20, 30)}

This is the flexibility that dictionaries offer, allowing you to store a variety of data types as values.

Dictionary Methods

‚óè keys()

The keys() method returns a list of all keys in a dictionary.
For example:
students = {2: 'Reena', 3: 'Aman', 4: 'Charlie'}


‚óè values()
students = {2: 'Reena', 3: 'Aman', 4: 'Charlie'}
The values() method returns a list of all values in the dictionary.

For example:
students = {2: 'Reena', 3: 'Aman', 4: 'Charlie'}


‚óè items()

The items() method returns a list of tuples containing key-value pairs.
For example:
students = {2: 'Reena', 3: 'Aman', 4: 'Charlie'}


‚óè copy()

The copy() method creates a copy of the dictionary.

For example:
students = {2: 'Reena', 3: 'Aman', 4: 'Charlie'}


üéØ This method does not modify the original dictionary; it creates a separate copy.

Looping Through a Dictionary

‚óè Using keys()

You can loop through all the keys in a dictionary using the keys() method.
Example:
students = {2: 'Reena', 3: 'Aman', 4: 'Charlie'}


‚óè Using values()

You can loop through all the values in a dictionary using the values() method.
Example:
students = {2: 'Reena', 3: 'Aman', 4: 'Charlie'}


‚óè Using items()
You can loop through both keys and values in a dictionary using the items() method.
Example:
students = {2: 'Reena', 3: 'Aman', 4: 'Charlie'}


üéØ This allows you to access both keys and values in a single loop.
